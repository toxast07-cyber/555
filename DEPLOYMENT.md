# Руководство по развертыванию

## Предварительные требования

- Visual Studio 2022 (Community, Professional или Enterprise)
- .NET 6.0 SDK или выше
- Windows 10/11

## Этапы установки

### 1. Клонирование репозитория
```bash
git clone https://github.com/toxast07-cyber/555.git
cd 555
```

### 2. Восстановление зависимостей
```bash
dotnet restore
```

### 3. Построение проекта
```bash
dotnet build
```

### 4. Запуск приложения
```bash
dotnet run --project UtilityBilling
```

## Запуск тестов

```bash
dotnet test
```

## Структура приложения

### Основные компоненты:

1. **Models** - Модели данных
   - `Utility.cs` - Модель коммунальной услуги
   - `BillingRecord.cs` - Модель счета

2. **Services** - Бизнес-логика
   - `UtilityService.cs` - Сервис управления услугами

3. **Forms** - Windows Forms интерфейсы
   - `MainForm.cs` - Главное окно приложения
   - `BillingForm.cs` - Форма создания счета
